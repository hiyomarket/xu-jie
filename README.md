## 📌 已確認的核心遊戲機制

### ✅ 元素系統（火/風/地/水）
- 火 → 風 → 地 → 水 → 火（循環剋制）
- 剋制倍率：+30%

### ✅ 角色屬性計算
- 直接參考魔力寶貝原版公式（網路可查）
- 不自行發明，確保數值平衡有據可循

### ✅ 抓寵系統（簡化版封印捕捉）
- 所有玩家皆可捕捉（廢除封印師職業限定）
- 通用捕捉器道具（不分種族）
- 圖鑑改為「遇見過」門檻
- 血量 ≤ 30% 時捕捉率最高
- 保留：寵物檔數/資質系統
- 保留：變異寵機制

### ✅ 經濟三層制
- 靈石（Premium，台幣付費）
- 遊戲幣（遊戲內產出）
- 積分值（免費/付費混合）
