# 墟界 ── 升級與成長系統 v1.0
> 版本：v1.0｜日期：2026-03-29｜狀態：✅ Boss已確認
> 依據：魔力寶貝模型 + 墟界五行屬性

---

## 0. 核心原則

```
1. 人物和寵物都有「境界」和「五行」
2. 人物升級：自由分配點數
3. 寵物升級：跟隨主人戰鬥自動獲得經驗
4. 五行屬性影響：寵物天生五行 vs 主人命格
```

## 3. 寵物升級系統（方案A：跟隨主人）

### 3.1 核心原則

```
主人戰鬥 → 寵物自動獲得經驗 → 寵物自動升級
寵物無需單獨操作
```

### 3.2 寵物經驗獲取

| 條件 | 經驗倍率 |
|------|---------|
| 寵物出戰（擔任主力）| 100% 經驗 |
| 寵物待命（不上場）| 30% 經驗 |

### 3.3 寵物每次升級

```
寵物每次升級：+3 點「潛能點數」（比人物少）
分配至：HP / ATK / DEF / SPD / MATK（5項，無MP）
```

| 屬性 | 每點效果 |
|------|---------|
| HP | +15 HP |
| ATK | +2 ATK |
| DEF | +2 DEF |
| SPD | +1.5 SPD |
| MATK | +2 MATK |

### 3.4 寵物境界上限

```
主人的境界影響寵物可達到的最高等級：
主人境界：初界 → 寵物最高Lv.30
主人境界：中界 → 寵物最高Lv.60
主人境界：高界 → 寵物最高Lv.99
```

## 4. 寵物五行屬性

### 4.1 寵物的五行來源

```
寵物天生具有自己的五行屬性
但會「繼承主人的五行比例」做為成長加成
寵物的五行影響：
  - 屬性剋制（戰鬥時）
  - 可學習的技能（對應五行）
  - 與主人的命格是否相合
```

> 例：主人的火注靈多 → 寵物的火屬性加成更高
> 這就是「主人與寵物共振」的核心概念

### 4.2 五行剋制

```
木→土→水→火→金→木（循環剋制）
剋制：+20% 傷害
被剋：-20% 傷害
同屬：無增減
```

### 4.3 命格相合效果

| 相合關係 | 效果 |
|---------|------|
| 主人命格 = 寵物五行 | 寵物經驗+20% |
| 主人命格 剋 寵物五行 | 寵物經驗-10% |
| 寵物五行 剋 主人命格 | 寵物經驗-10% |

## 7. 魔物境界基礎值公式（企劃顧問提供）

### 7.1 境界倍率

| 境界 | 倍率 | 適用 |
|------|------|------|
| 初界 | ×1.0 | 野外普通怪 |
| 中界 | ×1.5 | 精英怪 |
| 高界 | ×2.0+ | BOSS級 |

### 7.2 境界基礎值公式

```
最終ATK = 等級基礎值 × 境界倍率 + (主五行點數 × 5%)
最終HP  = 等級基礎值 × 境界倍率 + (主五行點數 × 5%)
```

### 7.3 五行屬性點數加成

```
主五行屬性點數 × 5% = 額外屬性加成
例：火焰鼠（火點8）× 5% = +40% ATK加成
```

## 8. 精神值（SPR）系統（依據魔力寶貝研究新增）

### 8.1 精神值說明

```
精神值（SPR）是魔法攻防的獨立計算軸
類似魔力寶貝的「精神」屬性
影響：魔法攻擊力、魔法防禦力
```

### 8.2 精神值來源

```
由「水」屬性主宰
水 → MP + 精神值
每次水屬性點數分配：+2 精神值
```

### 8.3 精神值公式

```
MATK = Lv×8 + 15 + (火÷100)×25 + (水÷100)×10   （原有火加成 + 水精神加成）
MDEF = Lv×6 + 10 + (土÷100)×20 + (水÷100)×10   （原有土加成 + 水精神加成）
```

## 9. 回復力（HEAL）系統（依據魔力寶貝研究新增）

### 9.1 回復力說明

```
回復力（HEAL）影響治療技能的治療量
由「木」屬性主宰（木 → HP + HEAL）
```

### 9.2 治療公式

```
單體治療量 = 技能等級 × 60 × (回復力%)
強群體治療量 = 技能等級 × 36 × (回復力%)
超強群體治療量 = 技能等級 × 24 × (回復力%)
```

寵物的所有屬性均由 BP 衍生而來
