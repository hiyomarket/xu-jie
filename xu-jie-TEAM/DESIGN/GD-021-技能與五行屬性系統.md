## 2. 魔力寶貝技能系統分析

### 2.1 技能分類

```
┌─────────────────────────────────────────────────────────┐
│                 魔力寶貝技能分類                        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  【戰鬥技能】                                           │
│  ├── 戰鬥攻擊技能（物理）                                │
│  ├── 戰鬥攻擊技能（魔法）                                │
│  ├── 戰鬥防禦技能                                       │
│  ├── 戰鬥輔助技能                                       │
│  └── 狀態變化技能                                       │
│                                                         │
│  【生產技能】                                           │
│  ├── 採集技能                                          │
│  ├── 製作技能                                          │
│  ├── 修理技能                                          │
│  └── 鑑定技能                                          │
│                                                         │
│  【寵物技能】                                           │
│  ├── 先天技能（捕捉自帶）                                │
│  ├── 可學習技能（戰鬥中使用）                           │
│  └── 被動技能                                          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 2.2 學習方式

| 學習方式 | 說明 | 範例 |
|----------|------|------|
| **NPC 學習** | 找特定 NPC 對話，花費魔幣 | 基礎戰鬥技能 100 魔幣 |
| **任務獲得** | 完成特定任務獲得 | 職業任務獎勵 |
| **技能欄位** | 每個技能消耗 1-3 格欄位，共 10 格 | 乾坤一擲消耗 2 格 |
| **忘記技能** | 付出 50% 費用可忘記，空出欄位 | 重新學習其他技能 |

### 2.3 技能等級

```
技能經驗累積 → 技能升級
• 1-5 級：戰鬥中使用累積經驗
• 6-10 級：需要透過任務或修煉
• 使用得意技（本职技能）得 2 倍經驗
```

```
┌─────────────────────────────────────────────────────────┐
```

```
│                                                         │
│  Step 3: 習得成功                                       │
│  ├── 消耗技能欄位                                       │
│  ├── 開始累積技能經驗                                  │
│  └── 可升級（最高 Lv.10）                               │
│                                                         │

```

### 3.3 技能欄位設計

| 項目 | 數量 | 說明 |
|------|------|------|
| 初始技能欄 | 8 格 | 免費 |
| 最大技能欄 | 12 格 | VIP/任務擴充 |
| 技能消耗 | 1-3 格/技能 | 厲害技能消耗多 |

### 3.4 技能忘記機制

```
忘記費用 = 原價 × 50%
忘記後還原 50% 熟練度
可隨時重新學習
```

## 4. 五行屬性分配系統（墟界獨家設計）

### 4.1 靈感來源：魔力寶貝寵物屬性分布

```
魔力寶貝範例：火焰鼠
• 檔位：火 9 + 地 1
• 總檔位：10 檔
• 影響：火屬性攻擊強、地屬性攻擊弱
```

### 4.2 墟界五行屬性分配設計

#### 4.2.1 屬性點數來源

```
每次升級獲得：5 點屬性點數

分配方式：
├── 自由分配（玩家自己決定）
└── 或 │
       ├── 根據當前命格傾向自動分配
       └── 根據武器/裝備傾向
```

#### 4.2.2 屬性點數分配表

```
┌─────────────────────────────────────────────────────────┐
│                 升級屬性點數分配（每級 5 點）           │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  方案 A：自由分配                                      │
│  ──────────────                                        │
│  玩家自由分配到五個屬性                                 │
│  每個屬性 0-5 點                                        │
│                                                         │
│  方案 B：命格傾向（自動分配）                           │
│  ─────────────────                                      │
│  木命 → 木 +2, 其他 +1                                │
│  火命 → 火 +2, 其他 +1                                │
│  土命 → 土 +2, 其他 +1                                │
│  金命 → 金 +2, 其他 +1                                │
│  水命 → 水 +2, 其他 +1                                │
│                                                         │
│  方案 C：自訂分配                                       │
│  ──────────────                                        │
│  玩家可以選擇「主流」屬性                              │
│  例如：火 +3, 土 +1, 金 +1                            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 4.2.3 五行屬性對技能學習的影響

```
┌─────────────────────────────────────────────────────────┐
│              五行屬性 → 技能學習門檻                    │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  🪵 木屬性點數 → 學習恢復系技能需木 ≥ 20               │
│  🔥 火屬性點數 → 學習攻擊魔法需火 ≥ 20                 │
│  🪨 土屬性點數 → 學習防御系技能需土 ≥ 20               │
│  ⚔️ 金屬性點數 → 學習物理攻擊需金 ≥ 20                 │
│  💧 水屬性點數 → 學習輔助技能需水 ≥ 20                 │
│                                                         │
│  ─────────────────────────────────────────              │
│                                                         │
│  高階技能門檻（例如 Lv.6+）：                           │
│  需要該屬性 ≥ 50 點                                     │
│                                                         │
│  覺醒技能（例如 Lv.9-10）：                             │
│  需要該屬性 ≥ 80 點                                     │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 4.3 屬性點數查詢表

```
┌─────────────────────────────────────────────────────────┐
│                 屬性點數與加成對照                      │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  每 10 點屬性 = +3% 相關技能威力                       │
│                                                         │
│  範例：火屬性 30 點                                     │
│  → 火系魔法攻擊威力 +9%                                │
│  → 可學習「火焰箭」（需火 20）                         │
│  → 不可學習「烈焰風暴」（需火 50）                      │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 6. 寵物技能系統

### 6.1 寵物屬性分布（參考魔力寶貝檔位）

```
┌─────────────────────────────────────────────────────────┐
│              寵物五行屬性分布系統                        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  捕捉時隨機生成五行屬性分布                              │
│  總和 = 100 點                                          │
│                                                         │
│  範例：                                                  │
│  ├── 火焰鼠：火 80 + 土 10 + 水 5 + 木 3 + 金 2       │
│  ├── 綠芽史萊姆：木 70 + 水 20 + 火 5 + 土 3 + 金 2   │
│  └── 鐵甲熊：土 60 + 金 30 + 木 5 + 火 3 + 水 2       │
│                                                         │
│  檔位評價：                                              │
│  ├── 0-2 檔：極品（某屬性集中 80%+）                    │
│  ├── 3-5 檔：普通（均衡分布）                          │
│  └── 6-10 檔：較差（平均分配，無特色）                  │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 6.2 寵物技能學習

```
┌─────────────────────────────────────────────────────────┐
│              寵物技能學習流程                           │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  【先天技能】                                           │
│  捕捉時自帶，不可移除                                    │
│  數量：1-3 個                                          │
│                                                         │
│  【可學習技能】                                         │
│  與人物技能類似，但有所不同：                            │

│  └── 升級：戰鬥中使用累積經驗                            │

│  【天賦技能】（天命技能）                                │
│  寵物專屬的強大技能，可升級                              │
```

## 8. 實作重點

### 8.1 需要新增的資料表

```sql
-- 玩家屬性點數分配
CREATE TABLE player_attribute_points (
  id INT PRIMARY KEY AUTO_INCREMENT,
  user_id INT NOT NULL,
  wood_points INT DEFAULT 0,
  fire_points INT DEFAULT 0,
  earth_points INT DEFAULT 0,
  metal_points INT DEFAULT 0,
  water_points INT DEFAULT 0,
  total_points INT DEFAULT 0,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 寵物五行屬性分布
CREATE TABLE pet_element_distribution (
  id INT PRIMARY KEY AUTO_INCREMENT,
  pet_id INT NOT NULL,
  wood_ratio INT DEFAULT 0,
  fire_ratio INT DEFAULT 0,
  earth_ratio INT DEFAULT 0,
  metal_ratio INT DEFAULT 0,
  water_ratio INT DEFAULT 0,
  grade INT DEFAULT 5,  -- 檔位評價
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 8.2 需要修改的系統

| 項目 | 修改內容 |
|------|----------|
| 升級系統 | 每次升級發放 5 點屬性點數 |
| 技能學習 | 新增屬性門檻檢查 |
| 戰鬥計算 | 新增屬性加成 |
