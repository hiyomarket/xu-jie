# 墟界 ── 遊戲參數後台管理系統（GM Tool）
> 設計版本：v1.0｜2026-03-28｜由大管家依Boss指令編寫

---

## 1. 設計目的

提供一個完整的 **GM 後台**，讓運營人員可以即時調整所有影響角色成長的數值參數，
無需修改代碼即可即時生效，平衡遊戲生態。

---

## 3. 後台功能模組

### 3.1 公式模擬器（最重要）

即時輸入：
- 等級
- 各屬性注靈值
- 裝備檔位（白/綠/藍/紫/橙）
- 對應Boss

輸出：
- 完整角色屬性面板
- 預估DPS
- 對Boss存活回合數
- 與Boss的平衡評級（S/A/B/C/D）

### 3.2 參數調節器

以 **表格+拉霸** 形式呈現所有參數，
每次修改即時預覽對Lv.50/Lv.70/Lv.99玩家的影響。

支援：
- 單一參數調整
- 批量套用（例：所有Boss ATK × 0.8）
- 版本記錄（可還原）

### 3.3 Boss 試算表

對所有Boss進行：
- Lv.40 / Lv.55 / Lv.70 單人和組隊實戰模擬
- 顯示「可挑戰 / 需治療 / 不可能」標記

### 3.4 平衡警報

當參數調整後，系統自動檢測：
- 某Lv範圍玩家是否對某Boss完全無法取勝
- DPS 是否超出合理範圍（< 5回合 或 > 60回合）
- 自動標記異常值

## 4. 技術規格

- **前端**：React + TailwindCSS（或直接用 HTML + Vanilla JS）
- **後端**：Node.js / Express（或純前端 JSON 設定檔）
- **資料儲存**：JSON 設定檔（易於 Git 版控）
- **部署**：Vultr VPS（同一台遊戲伺服器）
- **取用權限**：GM 等級以上帳號
