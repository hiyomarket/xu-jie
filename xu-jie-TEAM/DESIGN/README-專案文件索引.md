## 📌 當前共識狀態

### ✅ 已確認（系統可直接使用）
- 玩家屬性公式（含五行抗性）：HP = Lv×12+80 + 木注靈×0.3 + 裝備
- ATK = Lv×8+15 + 火注靈×0.2 + 裝備
- 抗性 = min(50%, 對應注靈÷20)
- 傷害公式：ATK×1.5 - DEF×0.5 × (1-抗性%)
- 死亡騎士：HP=25000, ATK=200, DEF=200, SPD=45
- 三人組隊是主要挑戰模式
- 14隻Boss屬性（見GD-022）

### ⏳ 待裁決
- 太岳磐石巨靈DEF=720是否調降至400-500
- 衝裝系統是否立項
- 鬼公子・噩是否列入正式Boss
- Vultr新VPS（IP+SSH密碼）
- xujie.io申請進度
