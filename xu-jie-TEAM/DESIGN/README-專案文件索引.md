## 🎯 GD-028 核心內容摘要（戰鬥系統根基）

### 人物屬性公式
```
HP  = Lv×12 + 80 + (木÷100)×30
MP  = Lv×8  + 40 + (水÷100)×20
ATK = Lv×8  + 15 + (火÷100)×30
DEF = Lv×8  + 15 + (土÷100)×30
SPD = Lv×6  + 10 + (金÷100)×20
MATK= Lv×8  + 15 + (火÷100)×25 + (水÷100)×10
MDEF= Lv×6  + 10 + (土÷100)×20 + (水÷100)×10
暴擊傷害 = 150 + (火÷100)×10
暴擊率   = 5   + (金÷100)×2
```

### 五行→屬性負責
```
木 → HP、HEAL（回復力）
火 → ATK、MATK、暴擊傷害
土 → DEF、MDEF
金 → SPD、HIT、暴擊率
水 → MP、精神值（SPR）
```

### 五行剋制
```
木→土→水→火→金→木（+20%/-20%）
```

### 種族相剋（兩條封閉循環鏈）
```
【鏈1】人 → 野 → 植 → 不 → 人
【鏈2】龍 → 飛 → 昆 → 特 → 金 → 龍
```

### 寵物BP系統（每級+3點）
```
體力BP → HP、HEAL
力量BP → ATK
強度BP → DEF
敏捷BP → SPD
魔法BP → MP、MATK（SPR）
```

### 境界倍率
```
初界×1.0 / 中界×1.5 / 高界×2.0+
主五行點數 × 5% = 額外加成
```

### 治療公式
```
單體治療量 = 等級 × 60 × 回復力%
```

### 狀態異常
```
中毒 / 石化 / 昏睡 / 混亂 / 遺忘
抵抗方式：對應屬性越高，持續時間越短
```
