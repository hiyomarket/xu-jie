# 墟界 ── 全域屬性系統統一規格書 v1.0
> 版本：v1.0｜日期：2026-03-28｜狀態：正式標準
> 用途：提供系統Agent重建遊戲資料庫的完整依據

---

## 0. 核心共識（所有人員必須遵守）

```
所有遊戲角色（玩家/怪物/Boss/寵物）皆使用同一套屬性框架
差異僅在於「數值來源」不同：
  玩家 → 等級基礎 + 注靈 + 裝備
  怪物 → 等階設定值（設計師指定）
  Boss → 境界強度表（設計師指定）
  寵物 → 繼承玩家屬性的固定比例
```

---

## 1. 統一屬性欄位

所有角色皆具備以下屬性欄位：

| 欄位 | 類型 | 說明 |
|------|------|------|
| `hp` | int | 生命值（無上限）|
| `mp` | int | 魔法值（無上限）|
| `atk` | int | 物理攻擊力 |
| `def` | int | 物理防禦力 |
| `spd` | int | 速度（影響行動順序）|
| `matk` | int | 魔法攻擊力 |
| `mdef` | int | 魔法防禦力 |
| `wood_res` | int | 木屬性抗性（%）|
| `fire_res` | int | 火屬性抗性（%）|
| `earth_res` | int | 土屬性抗性（%）|
| `metal_res` | int | 金屬性抗性（%）|
| `water_res` | int | 水屬性抗性（%）|
| `element` | string | 自身屬性（木/火/土/金/水）|

---

## 2. 傷害公式（所有情境統一）

```
最終傷害 = max(1, int(ATK × 1.5 - DEF × 0.5)) × (1 - 目標抗性% / 100)

若為魔法傷害：
最終傷害 = max(1, int(MATK × 1.5 - MDEF × 0.5)) × (1 - 目標抗性% / 100)

屬性剋制（額外±20%）：
木 → 土（+20%）、火（-20%）
火 → 金（+20%）、水（-20%）
土 → 水（+20%）、木（-20%）
金 → 木（+20%）、火（-20%）
水 → 火（+20%）、土（-20%）
```

## 3. 玩家屬性系統

### 3.1 等級成長公式

```
HP_base  = Lv × 12 + 80
MP_base  = Lv × 8 + 40
ATK_base = Lv × 8 + 15
DEF_base = Lv × 8 + 15
SPD_base = Lv × 6 + 10
MATK_base = Lv × 8 + 15
MDEF_base = Lv × 6 + 10
```

### 3.2 注靈系統（上限1000）

| 注靈類型 | 每100點效果 |
|---------|------------|
| 木 | HP +30 / 木抗 +5% |
| 火 | ATK +20 / 火抗 +5% |
| 土 | DEF +20 / 土抗 +5% |
| 金 | SPD +15 / 金抗 +5% |
| 水 | MP +20 / 水抗 +5% |

### 3.3 裝備加成（紫色套裝為例）

|------|-----|-----|-----|-----|------|
| 武器 | +100 | — | — | — | +70 |
| 防具 | — | +80 | +200 | — | — |
| 飾品 | +20 | +20 | +100 | +25 | — |

## 4. 怪物屬性系統

### 4.1 設計原則
- **不做等級成長**：怪物屬性由設計師直接指定
- **保留五屬性**：每隻怪物皆有屬性（`element` 欄位）
- **保留五行抗性**：每隻怪物有5項抗性值（預設0%）
- **抗性計算**：與玩家同級（最大50%）

### 4.2 怪物屬性來源
- 境界系數 × 等階基礎值
- 境界分為：初界/中界/中高界/高界
- 同時影響HP倍率、ATK倍率

### 4.3 境界強度對照

| 境界 | HP倍率 | ATK倍率 | 說明 |
|------|--------|---------|------|
| 初界 | ×1 | ×1 | Lv.1-20 |
| 中界 | ×2 | ×1.5 | Lv.21-40 |
| 中高界 | ×4 | ×2.5 | Lv.41-60 |
| 高界 | ×8 | ×4 | Lv.61-99 |

### 4.4 怪物五行抗性計算

```
怪物抗性 = 根據怪物屬性自動給予基礎抗性
木屬怪物：木抗+20%，被土屬克制（火抗-10%）
火屬怪物：火抗+20%，被水屬克制（木抗-10%）
土屬怪物：土抗+20%，被木屬克制（金抗-10%）
金屬怪物：金抗+20%，被火屬克制（土抗-10%）
水屬怪物：水抗+20%，被土屬克制（水抗-10%）
```

## 5. Boss屬性系統

### 5.1 Boss與怪物的差異
- Boss擁有更高的**境界倍率**
- Boss擁有**技能欄位**（被動/主動/大招）
- Boss的**五行抗性可達100%**（不設上限視覺效果）
- Boss有**稱號/描述/Lore**

### 5.2 境界倍率

| 境界 | HP倍率 | ATK倍率 | 適用 |
|------|--------|---------|------|
| 初界 | ×3 | ×2 | 小王 |
| 中界 | ×6 | ×4 | 中王 |
| 中高界 | ×12 | ×8 | 大王 |
| 高界 | ×24 | ×16 | 區域守護者 |

### 5.3 Boss抗性設計
- **基礎抗性**：屬性相同+50%（非疊加，取最高）
- **抗性上限**：200%（可被完全剋制，但很難）
- **階級加成**：等階越高，抗性基礎越高
  - T1：基礎抗性+10%
  - T2：基礎抗性+20%
  - T3：基礎抗性+30%

### 5.4 死亡騎士屬性（代碼：MOB_DEATH_KNIGHT）

```
名稱：死亡騎士（送葬者）
五行屬性：土
等級：50
HP：25000
ATK：200
DEF：200
SPD：45
MATK：180
MDEF：130
土抗：50%（土屬+20% + T2等階+20%）
火抗：20%（被火剋制）
木抗：10%（剋制土）
HP上限抗性：各屬性抗性均為0%
```

## 6. 寵物屬性系統

### 6.1 寵物屬性來源
- 寵物不繼承玩家注靈，獨立計算屬性
- 寵物有自己的**五行屬性**和**等級**
- 寵物升級方式：與玩家共同戰鬥獲得經驗

### 6.2 寵物屬性公式

```
HP  = 主人HP × 0.6 + 寵物Lv × 5
MP  = 主人MP × 0.4
ATK = 主人ATK × 0.35 + 寵物Lv × 3
DEF = 主人DEF × 0.4 + 寵物Lv × 2
SPD = 主人SPD × 0.5 + 寵物Lv × 2
MATK= 主人MATK × 0.3 + 寵物Lv × 3
MDEF= 主人MDEF × 0.3 + 寵物Lv × 2
五行屬性：根據寵物種類設定（固定）
五行抗性：根據寵物屬性自動計算（與怪物相同）
```

### 6.3 寵物戰鬥行為
- 每回合行動一次（獨立於主人）
- 行動類型：攻擊 / 防禦 / 治療主人 / 待機
- AI自動選擇，忠誠度影響行動質量

### 6.4 寵物五行抗性（固定值）

```
木系寵物：木抗30%、火抗-10%
火系寵物：火抗30%、水抗-10%
土系寵物：土抗30%、木抗-10%
金系寵物：金抗30%、火抗-10%
水系寵物：水抗30%、土抗-10%
```

## 7. 實戰計算範例

### 7.1 玩家 vs Boss（無屬性剋制）
```
玩家：ATK=615, 火抗=25%
Boss：ATK=200, 火抗=50%

我打Boss：max(1, 615×1.5 - 200×0.5) = 872 damage
Boss打我（火屬）：max(1, 200×1.5 - 350×0.5) × (1-25/100)
  = max(1, 300-175) × 0.75 = 125 × 0.75 = 93 damage ✅
```

### 7.2 玩家 vs Boss（火剋金，屬性劣勢）
```
玩家：金屬性，ATK=615, 火抗=0%
Boss：火屬性，ATK=200, 火抗=50%

Boss打我（屬性劣勢）：
  火剋金 → 額外+20%傷害
  = max(1, 200×1.5 - 350×0.5) × 1.2 × (1-0/100)
  = 125 × 1.2 = 150 damage ⚠️
```

## 8. 資料庫欄位對照

### 8.1 game_player_stats（玩家）
```sql
lv, hp, mp, atk, def, spd, matk, mdef,
wood_infusion, fire_infusion, earth_infusion, metal_infusion, water_infusion,
wood_res, fire_res, earth_res, metal_res, water_res,
element  -- 預設「無」，玩家可透過天命改變屬性
```

### 8.2 game_monster_catalog（怪物）
```sql
code, name, element, lv,
hp, atk, def, spd, matk, mdef,
wood_res, fire_res, earth_res, metal_res, water_res,
is_boss, tier, realm, skills[], description
```

### 8.3 game_pet_catalog（寵物）
```sql
code, name, element, lv,
hp, atk, def, spd, matk, mdef,
wood_res, fire_res, earth_res, metal_res, water_res,
master_hp_ratio, master_atk_ratio, master_def_ratio, master_spd_ratio,
skill_pool[], loyalty_max
```
