# 墟界 ── 全域屬性系統統一規格書 v1.0
> 版本：v1.0｜日期：2026-03-28｜狀態：正式標準
> 用途：提供系統Agent重建遊戲資料庫的完整依據

---

## 0. 核心共識

```
所有遊戲角色（玩家/怪物/Boss/寵物）皆使用同一套屬性框架
差異僅在於「數值來源」不同：
  玩家 → 等級基礎 + 注靈 + 裝備
  怪物 → 等階設定值（設計師指定）
  Boss → 境界強度表（設計師指定）
  寵物 → 繼承玩家屬性的固定比例
```

---

## 1. 統一屬性欄位

| 欄位 | 類型 | 說明 |
|------|------|------|
| `hp` | int | 生命值（無上限）|
| `mp` | int | 魔法值（無上限）|
| `atk` | int | 物理攻擊力 |
| `def` | int | 物理防禦力 |
| `spd` | int | 速度（影響行動順序）|
| `matk` | int | 魔法攻擊力 |
| `mdef` | int | 魔法防禦力 |
| `wood_res` | int | 木屬性抗性（%，最大50%）|
| `fire_res` | int | 火屬性抗性（%，最大50%）|
| `earth_res` | int | 土屬性抗性（%，最大50%）|
| `metal_res` | int | 金屬性抗性（%，最大50%）|
| `water_res` | int | 水屬性抗性（%，最大50%）|
| `element` | string | 自身屬性（木/火/土/金/水）|

---

## 2. 傷害公式

```
物理傷害 = max(1, int(ATK × 1.5 - DEF × 0.5)) × (1 - 抗性 / 100)
魔法傷害 = max(1, int(MATK × 1.5 - MDEF × 0.5)) × (1 - 抗性 / 100)

屬性剋制：
  木→土（+20%）、火→金（+20%）、土→水（+20%）、金→木（+20%）、水→火（+20%）
  被剋制：-20%
```

## 3. 玩家屬性系統（最終版）

### 等級成長公式
```
HP   = Lv × 12 + 80
MP   = Lv × 8 + 40
ATK  = Lv × 8 + 15
DEF  = Lv × 8 + 15
SPD  = Lv × 6 + 10
MATK = Lv × 8 + 15
MDEF = Lv × 6 + 10
```

### 注靈加成（上限1000）
| 注靈 | 每100點效果 |
|-----|------------|
| 木 | HP +30 / 木抗 +5% |
| 火 | ATK +20 / 火抗 +5% |
| 土 | DEF +20 / 土抗 +5% |
| 金 | SPD +15 / 金抗 +5% |
| 水 | MP +20 / 水抗 +5% |

### 五行抗性公式（最終版）
```
抗性 = min(50%, 對應屬性注靈值 ÷ 20)

例：火注靈=1000 → 火抗 = min(50%, 1000÷20) = 50%
例：火注靈=500  → 火抗 = min(50%, 500÷20)  = 25%
```

### 裝備加成（紫色套裝）
| 部位 | ATK | DEF | HP | SPD | MATK |
|------|-----|-----|-----|-----|------|
| 武器 | +100 | — | — | — | +70 |
| 防具 | — | +80 | +200 | — | — |
| 飾品 | +20 | +20 | +100 | +25 | — |

### Lv.50 含紫色套裝
| 檔位 | HP | ATK | DEF | 火抗 |
|------|-----|-----|-----|------|
| 無注靈 | 980 | 515 | 495 | 0% |
| 中(500) | 1,130 | 615 | 595 | 25% |
| 滿(1000) | 1,280 | 715 | 695 | 50% |

## 4. 怪物屬性系統

### 設計原則
- 不做等級成長，屬性由設計師直接指定
- 每隻怪物有 `element` 欄位
- 每隻怪物有5項抗性（基礎0%）

### 境界倍率
| 境界 | HP倍率 | ATK倍率 | 適用 |
|------|--------|---------|------|
| 初界 | ×3 | ×2 | 小王 |
| 中界 | ×6 | ×4 | 中王 |
| 中高界 | ×12 | ×8 | 大王 |
| 高界 | ×24 | ×16 | 區域守護者 |

### 抗性計算
```
怪物抗性 = 屬性基礎抗性（+20%~50%）+ 等階加成（T1+10% / T2+20% / T3+30%）
例：土屬怪物T2 → 土抗 = 基礎20% + T2加值20% = 40%
```

## 5. Boss屬性系統

### 死亡騎士（MOB_DEATH_KNIGHT）
```
HP=25000 / ATK=200 / DEF=200 / SPD=45 / MATK=180 / MDEF=130
五行屬性：土
土抗=50%（屬性20% + T2加值20%）/ 火抗=20%（被火剋制）
木抗=10%（剋制土）
```

## 6. 寵物屬性系統

### 屬性公式
```
HP   = 主人HP × 0.6 + 寵物Lv × 5
ATK  = 主人ATK × 0.35 + 寵物Lv × 3
DEF  = 主人DEF × 0.4 + 寵物Lv × 2
SPD  = 主人SPD × 0.5 + 寵物Lv × 2
```

## 7. 實戰範例

```
玩家（Lv.50，火注靈500）：ATK=615，火抗=25%
Boss（土屬，T2）：ATK=200，DEF=200，火抗=50%

我打Boss：max(1, 615×1.5 - 200×0.5) = 872
Boss打我（火剋土+20%）：872 × 1.2 × (1-25/100) = 872 × 1.2 × 0.75 = 785 ✅
```
