## 1. 概述

本設計書參考魔力寶貝的寵物系統，為墟界設計一套完整的寵物捕捉、培養、技能學習機制。寵物將作為玩家戰鬥的重要幫手，擁有獨立的三圍屬性和技能系統。

## 2. 寵物捕捉系統

### 2.1 捕捉機制

```
┌─────────────────────────────────────────────────────────┐
│                    寵物捕捉流程                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  戰鬥中觸發捕捉                                           │
│       │                                                 │
│       ▼                                                 │
│  玩家使用捕捉器                                          │
│       │                                                 │
│       ▼                                                 │
│  計算捕捉成功率                                          │
│  ────────────────────────                              │
│  成功率 = 基礎成功率 × (1 - 敵人HP%/100) × 捕捉器加成     │
│       │                                                 │
│       ▼                                                 │
│  成功 → 寵物入庫 / 失敗 → 繼續戰鬥                       │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 2.2 捕捉等級與檔位系統

| 項目 | 設計 | 說明 |
|------|------|------|
| **捕捉時等級** | 隨機 1-當前等級 | 例如Lv.10怪物可能抓到Lv.1-Lv.10的寵物 |
| **檔位（素質）** | 0-10檔 | 決定寵物三圍成長潛力 |
| **幸運捕捉** | 10%機率 | 有機會抓到完美檔位（0檔）寵物 |

### 2.3 檔位說明

```
檔位範圍：0檔（完美） ~ 10檔（最差）

| 檔位 | 評價 | 機率 | 三圍加成 |
|------|------|------|----------|
| 0檔 | 神寵/完美 | 2% | +30% |
| 1-2檔 | 極品 | 8% | +20% |
| 3-4檔 | 優秀 | 15% | +10% |
| 5-6檔 | 普通 | 25% | 標準 |
| 7-8檔 | 較差 | 30% | -10% |
| 9-10檔 | 廢寵 | 20% | -20% |

檔位影響：三圍（體/攻/防/敏/魔）
```

### 2.4 捕捉器設計

| 捕捉器 | 成功率加成 | 價格 | 來源 |
|--------|------------|------|------|
| 初級捕捉器 | +0% | 免費/商店 | 初始自帶 |
| 中級捕捉器 | +15% | 100 魔幣 | 商店 |
| 高級捕捉器 | +30% | 500 魔幣 | 商店 |
| 極品捕捉器 | +50% | 2000 魔幣 | BOSS掉落/商店 |
| 天命捕捉器 | +80% | 限時活動 | 活動限定 |

## 3. 寵物三圍屬性系統

### 3.1 五維屬性

```
┌─────────────────────────────────────────────────────────┐
│                    寵物五維系統                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  【體】體質 → HP上限                                      │
│  【攻】力量 → 物理攻擊力                                  │
│  【防】防禦 → 物理防禦力                                  │
│  【敏】敏捷 → 速度/閃避/命中                              │
│  【魔】魔力 → 魔法攻擊/治療量                             │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 3.2 屬性成長公式

```
每升一級獲得的屬性點數 = 基礎成長值 × (1 + 檔位加成%)

範例（Level 1 → Level 2）：
├── 0檔寵物：5點 × 1.30 = 6.5點
├── 5檔寵物：5點 × 1.00 = 5.0點
└── 10檔寵物：5點 × 0.80 = 4.0點
```

### 3.3 寵物類型與屬性傾向

| 類型 | 屬性傾向 | 特色 |
|------|----------|------|
| 獸系 | 體/攻 | 高HP、高攻擊 |
| 蟲系 | 攻/敏 | 高攻擊、高速度 |
| 龍系 | 攻/魔 | 物理/魔法兼備 |
| 植物系 | 體/防 | 高HP、高防禦 |
| 不死系 | 魔/防 | 魔法輸出 |
| 飛行系 | 敏/魔 | 高速度 |
| 特殊系 | 均衡 | 平衡型 |

## 4. 寵物技能系統

### 4.1 技能分類

```
┌─────────────────────────────────────────────────────────┐
│                    寵物技能分類                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  【先天技能】─ 捕捉時自带，不可移除                       │
│  ├── 攻擊技能                                           │
│  ├── 防禦技能                                           │
│  └── 輔助技能                                           │
│                                                         │
│  【天賦技能】─ 可學習，可替換，等同人物的天命技能         │
│  ├── 戰鬥強化                                           │
│  ├── 屬性加成                                           │
│  └── 特殊效果                                           │
│                                                         │
│  【被動技能】─ 永久生效                                   │
│  ├── 屬性抗性                                           │
│  └── 戰鬥加成                                           │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 4.2 先天技能（捕捉自帶）

每種寵物有 1-3 個先天技能，不可移除：

| 寵物範例 | 先天技能 |
|----------|----------|
| 綠芽史萊姆 | 撞擊、粘液攻擊 |
| 葉刃狐 | 爪擊、藤蔓束縛 |
| 幻影螳螂 | 刀刃亂舞、隱身 |
| 鐵木巨熊 | 猛擊、震懾 |

### 4.3 天賦技能（可學習）

玩家可以透過「天命系統」讓寵物學習技能，與人物技能類似：

| 技能類型 | 範例技能 | 學習方式 |
|----------|----------|----------|
| 攻擊強化 | 猛擊、連擊、暴擊 | 任務解鎖 |
| 防禦強化 | 護盾、鐵壁、反擊 | 任務解鎖 |
| 速度強化 | 加速、閃避、先制 | 任務解鎖 |
| 魔法強化 | 魔法加成、屬性魔法 | 任務解鎖 |
| 恢復系 | 小幅恢復、持續恢復 | 任務解鎖 |
| 吸血系 | 吸血、致命吸血 | 任務解鎖 |

### 4.4 天賦技能升級

```
天賦技能等級：Lv.1 ~ Lv.5

升級方式：類似人物技能
├── 戰鬥中使用累積經驗
├── 指定NPC強化
└── 消耗道具

每提升1級，效果+20%
```

## 6. 寵物戰鬥系統

### 6.1 寵物出戰

```
玩家戰鬥時可攜帶 1-3 隻寵物

出戰模式：
├── 主動出戰 → 寵物自動戰鬥
└── 被動援護 → 寵物在玩家HP<30%時出擊

寵物戰鬥屬性 = 基礎屬性 × (1 + 等級%/100)
```

### 6.2 寵物戰鬥行為

| 行為 | 觸發條件 | 效果 |
|------|----------|------|
| 攻擊 | 默認 | 造成物理/魔法傷害 |
| 防禦 | 被攻擊時 | 提升防禦 |
| 技能 | 有技能可用 | 使用天賦技能 |
| 護主 | 主人HP<30% | 援護攻擊 |
| 逃跑 | 必敗時 | 離開戰鬥（寵物不死亡） |

## 7. 任務系統（寵物天賦學習）

### 7.1 天賦技能任務

與人物技能類似，寵物需要透過任務解鎖天賦技能：

```
任務範例：猛擊的傳承
┌─────────────────────────────────────────────────────────┐
│ 任務名稱：猛擊的傳承                                     │
│ 任務類型：寵物專屬                                        │
│ 觸發NPC：寵物訓練師                                       │
│ ────────────────────────                                 │
│ 任務目標：                                               │
│ 1. 寵物等級達到 Lv.10                                    │
│ 2. 擊敗森林之王                                          │
│ 3. 收集「猛擊之證」                                       │
│ ────────────────────────                                 │
│ 任務獎勵：                                               │
│ ✓ 寵物習得「猛擊」天賦技能                               │
│ ✓ 寵物天賦技能欄 +1                                      │
└─────────────────────────────────────────────────────────┘
```

### 7.2 天賦技能欄擴充

| 欄位數 | 取得方式 |
|--------|----------|
| 2格 | 初始自帶 |
| +1格 | 任務「訓練師的認可」 |
| +1格 | 任務「天命之路」 |
| 最大5格 | VIP/商城 |

| 檔位 | 評價 | 總成長加成 | 單項加成範圍 |

| 7-8 | 較差 | -10% | 每項 -1~2 |
