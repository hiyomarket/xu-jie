# 墟界 ── 技能與成長系統全面重構（魔力寶貝模型）
> 版本：v1.0｜日期：2026-03-29｜狀態：初稿待裁決
> 依據：魔力寶貝經典系統 + 墟界五行屬性

---

## 0. 設計原則

```
1. 忘掉所有現有技能系統
2. 全面比照魔力寶貝：玩家技能靠「職業+升級」學習
3. 墟界額外多「五行屬性」：玩家有屬性寵物也有
4. BP值（戰鬥力）= 五行分配比例 × 種族值
```

## 3. 玩家五行屬性系統

### 3.1 核心差異：魔力寶貝玩家無屬性，墟界玩家有五行

魔力寶貝：玩家本身無屬性，只有寵物有屬性
墟界：玩家本身有五行屬性（由「命格」決定）

### 3.2 命格系統（五行屬性由命格決定）

| 命格 | 五行 | 效果 |
|------|------|------|
| 青龍命 | 木 | 生命氣息濃郁 |
| 朱雀命 | 火 | 戰意激昂 |
| 白虎命 | 金 | 銳意難擋 |
| 玄武命 | 水 | 靈力充沛 |
| 麒麟命 | 土 | 大地護持 |

### 3.3 玩家五行點數分配

```
每次升級：獲得 5 點「五行點數」
玩家可將點數分配到自己的命格屬性上
（類似魔力寶貝的「個體值/努力值」但更直觀）
```

| 分配 | 效果 |
|------|------|
| 投入木 | HP +5、命中+1 |
| 投入火 | ATK+3、MATK+3、暴擊+1 |
| 投入土 | DEF+3、MDEF+3、韌性+1 |
| 投入金 | SPD+2、HIT+2、暴率+1 |
| 投入水 | MP+5、抵抗+2 |

## 4. 怪物五行分配系統（BP值模型）

### 4.1 魔力寶貝的「種族值/個體值」概念

```
魔力寶貝：HP/ATK/DEF/SPD 為「種族值」(Base Stats)
           IV為「個體值」(0-31隨機)
           EV為「努力值」(0-252)

墟界 equivalent：
  種族值 = 怪物的「境界基礎值」（固定）
  個體值 = 怪物的「五行分配」（隨機）
  努力值 = 玩家培養（戰鬥獲得）
```

### 4.2 怪物五行分配（核心新系統）

每隻怪物有 **10點五行點數**，由系統隨機分配：

```
火9地1（火屬性9點，大地1點）
等於：HP種族值 × 0.9 + HP大地值 × 0.1
```

| 分配類型 | 效果 |
|---------|------|
| 火9地1 | HP偏低/ATK極高，剋水被土剋 |
| 火5地5 | 攻防均衡 |
| 水10 | HP極高/ATK偏低 |
| 木8火2 | HP高，ATK中等，木屬性攻 |

### 4.3 怪物五行分配表（範例）

| 怪物 | 木 | 火 | 土 | 金 | 水 | 總和 | 說明 |
|------|----|----|----|----|----|------|------|
| 火焰鼠 | 0 | 9 | 0 | 1 | 0 | 10 | 火9金1 |
| 水泥鼠 | 0 | 0 | 5 | 0 | 5 | 10 | 土5水5 |
| 金光龜 | 0 | 0 | 4 | 6 | 0 | 10 | 金6土4 |
| 葉龍 | 8 | 0 | 2 | 0 | 0 | 10 | 木8土2 |
| 混合妖 | 2 | 2 | 2 | 2 | 2 | 10 | 五行均等 |

### 4.4 五行點數影響結算公式

```
怪物最終ATK = 境界基礎ATK × (火點數/10) + 境界基礎ATK×0.5×(其他點/10)
怪物最終HP  = 境界基礎HP × (木點數/10) + 境界基礎HP×0.5×(其他點/10)
```

## 5. 技能與五行屬性的連動

### 5.1 技能屬性

每個技能有五行屬性，使用時受「屬性剋制」影響：

```
木→土→水→火→金→木
剋制：+20%傷害
被剋：-20%傷害
```

### 5.2 技能學習與五行親和

```
當玩家的「命格屬性」與技能屬性相同經驗值額外+50%
火命玩家學習火屬技能：經驗+50%
木命玩家學習木屬技能：經驗+50%
```

## 6. 升級點數系統（對照魔力寶貝）

### 6.1 每次升級獲得點數

```
每次升級：+5點「潛能點數」
玩家自由分配到六項基礎屬性
HP / MP / ATK / DEF / SPD / MATK
```

### 6.2 每點屬性的效果

| 屬性 | 每點效果 |
|------|---------|
| HP | +20 HP |
| MP | +10 MP |
| ATK | +3 ATK |
| DEF | +3 DEF |
| SPD | +2 SPD |
| MATK | +3 MATK |

### 6.3 Lv.99時的點數總量

```
99級 × 5點 = 495點（可自由分配）
全部投入HP：+9,900 HP
全部投入ATK：+1,485 ATK
```
