# 墟界 ── 技能與成長系統全面重構
> 版本：**v1.0 正式版**｜日期：2026-03-29｜狀態：✅ Boss已確認
> 依據：魔力寶貝經典系統 + 墟界五行屬性

---

## 0. 設計原則

```
1. 忘掉所有現有技能系統，全面比照魔力寶貝經典
2. 玩家技能靠「職業+升級」學習
3. 墟界比魔力寶貝多「五行屬性」維度
4. BP值（戰鬥力）= 五行分配比例 × 境界基礎值
```

---

## 1. 職業系統

### 1.1 起點：無業遊民

```
初始狀態：無職業、無技能
↓
達到 Lv.10：解鎖底部功能表「轉職行」
↓
選擇職業方向
↓
滿足轉職條件（等級/任務/轉職令）
↓
完成轉職試煉
↓
正式轉職 → 獲得新命格 + 新技能
```

| 職業 | 五行 | 擅長 | 初始技能 |

| 坦克 | 土 | 防禦/嘲諷 | 岩甲護盾 |

## 2. 技能學習系統

### 2.1 四種取得方式

| 方式 | 說明 |
|------|------|
| **升級自動** | 達到指定等級自動領悟（主要方式）|
| **職業解鎖** | 加入職業後自動獲得職業專屬被動技 |
| **技能書** | 使用技能書學習（商城購買/Boss掉落）|
| **天命覺醒** | 完成覺醒任務自動解鎖（天命/傳說技能）|

### 2.2 技能階級與五行門檻

| 階級 | 等級 | 五行門檻 | 取得方式 |
|------|------|----------|---------|
| 初階 | Lv.1-20 | ≥ 10點 | 升級自動 |
| 中階 | Lv.21-40 | ≥ 20-30點 | 升級自動 |
| 高階 | Lv.41-60 | ≥ 50點 | 技能書 |
| 天命 | Lv.61-80 | ≥ 80點 | 天命覺醒任務 |
| 傳說 | Lv.81-99 | ≥ 95點 | 傳說任務 |

### 2.3 技能類型

| 類型 | 說明 |
|------|------|
| 主動攻擊 | 對敵人造成傷害（消耗MP）|
| 主動輔助 | 增益/減益（消耗MP）|
| 被動技能 | 永久生效，無需操作 |
| 特殊技能 | 冷卻時間長，效果強大 |

## 3. 玩家命格系統

### 3.1 五大命格

| 命格 | 五行 | 效果 |
|------|------|------|
| 青龍命 | 木 | 生命氣息濃郁，HP/HEAL強化 |
| 朱雀命 | 火 | 戰意激昂，ATK/MATK強化 |
| 白虎命 | 金 | 銳意難擋，SPD/HIT/暴率強化 |
| 玄武命 | 水 | 靈力充沛，MP/抵抗強化 |
| 麒麟命 | 土 | 大地護持，DEF/MDEF強化 |

### 3.2 命格如何決定五行屬性

```
起始命格由系統根據玩家創角時的選擇決定
（選「火」系路線 → 朱雀命 → 火屬性）
命格決定後，該屬性的「五行點數」自動+10
```

### 3.3 命格對技能學習的影響

```
命格屬性技能：經驗值額外+50%
例：朱雀命（火）學習火球術 → 火屬技能經驗+50%
```

## 5. 怪物五行分配系統

### 5.1 對照魔力寶貝

| 魔力寶貝 | 墟界equivalent |
|---------|---------------|
| 種族值（Base Stats）| 境界基礎值（固定）|
| 個體值（IV）| 五行分配（隨機，10點制）|
| 努力值（EV）| 玩家培養（戰鬥獲得）|

### 5.2 怪物10點五行制

每隻怪物隨機分配 **10點** 五行點數：

| 火9地1 | 水10 | 木8火2 | 五行均等 |
|--------|------|--------|---------|
| ATK極高，HP低 | HP極高，ATK低 | HP高，攻防均衡 | 各項平均 |

### 5.3 實例

| 怪物 | 木 | 火 | 土 | 金 | 水 | 說明 |
|------|----|----|----|----|----|------|
| 火焰鼠 | 0 | 9 | 0 | 1 | 0 | 火9金1 |
| 水泥鼠 | 0 | 0 | 5 | 0 | 5 | 土5水5 |
| 金光龜 | 0 | 0 | 4 | 6 | 0 | 金6土4 |
| 葉龍 | 8 | 0 | 2 | 0 | 0 | 木8土2 |

### 5.4 屬性剋制

```
木→土→水→火→金→木（循環）
剋制：+20%傷害
被剋：-20%傷害
同屬：無增減
```
