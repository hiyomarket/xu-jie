## 一、系統總覽

墟界寵物系統以**魔力寶貝**為參照基準，設計一套完整的「捕捉 → 培養 → 技能 → 天命」循環。

```
┌──────────────────────────────────────────────────────────────┐
│                      墟界寵物系統循環                           │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  戰鬥中遭遇怪物                                              │
│       │                                                     │
│       ▼                                                     │
│  使用「獸魂甕」捕捉                                          │
│  成功率 = 基礎30% × HP因子 × 等級因子 × 道具加成              │
│       │                                                     │
│       ▼                                                     │
│  捕捉成功 → 獲得寵物（隨機等級 + 隨機檔位）                        │
│       │                                                     │
│       ▼                                                     │
│  培養：升級 → BP成長 → 屬性提升                               │
│       │                                                     │
│       ▼                                                     │
│  技能：天生技能（3格）+ 天命技能（3格，共6格）                      │
│       │                                                     │
│       ▼                                                     │
│  天命技能可升級（Lv1-10），與人物技能系統共享                       │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

## 二、寵物抓取系統

### 2.1 捕捉道具

| 道具名稱 | 稀有度 | 成功率加成 | 取得方式 |
|---------|---------|-----------|---------|
| 獸魂甕 | 普通 | +0%（基礎） | 野外怪物掉落（15%）、商店購買 |
| 秘銀獸魂甕 | 稀有 | +15% | 每日任務獎勵 |
| 星輝獸魂甕 | 傳說 | +30% | 月卡/季卡玩家每月領取3個 |
| 天命獸魂甕 | 限定 | +50% | 活動限定 |

> 與魔力寶貝封印卡邏輯相同，但簡化為4個品質階級。

### 2.2 捕捉率公式

```
捕捉成功率 (%) = 基礎捕捉率 × HP因子 × 等級因子 × 道具加成
```

| 參數 | 公式 | 說明 |
|------|------|------|
| **基礎捕捉率** | 30% | 對所有怪物適用的基準值 |
| **HP因子** | `(1 - CurrentHP/MaxHP)^0.8 × 0.7 + 0.3` | HP越低越好；HP為1%時因子≈0.78，HP為50%時因子≈0.54，HP為100%時因子=0.30 |
| **等級因子** | `1 / (1 + 0.02 × (MonsterLv - PlayerLv))` | 怪物高於玩家10等時，捕捉率降約17% |
| **道具加成** | 獸魂甕:×1.0 / 秘銀:×1.15 / 星輝:×1.30 / 天命:×1.50 | 捕捉器稀有度加成 |

**HP狀態 → 實際捕捉率速查（使用普通獸魂甕，玩家與怪物同級）：**

| HP狀態 | HP因子 | 捕捉成功率 |
|--------|--------|-----------|
| 100%（滿血） | 0.30 | **30%** |
| 75% | 0.41 | **41%** |
| 50% | 0.54 | **54%** |
| 25% | 0.69 | **69%** |
| 10% | 0.77 | **77%** |
| 1% | 0.78 | **78%** |

> ⚠️ **實作注意**：HP低於5%時怪物會進入「重傷」狀態，會自動停止攻擊並顯示捕捉確認視窗（而非直接死亡結算）。

### 2.3 捕捉時的等級分布

捕捉到的寵物等級服從**截斷正態分布**（以出現地點怪物平均等級μ為中位數，σ = μ/5）：

**分布表（以μ=怪物平均等級）：**

| 與μ的差距 | 出現機率 |
|-----------|---------|
| μ - 5 等以上 | 5% |
| μ - 4 ~ μ - 3 | 10% |
| μ - 2 ~ μ - 1 | 25% |
| μ（±0） | 35% |
| μ + 1 ~ μ + 2 | 15% |
| μ + 3 ~ μ + 4 | 7% |
| μ + 5 等以上 | 3% |

**實例：新手野外（μ=3，等級1~8範圍）**

| 等級 | 機率 |
|------|------|
| Lv1 | 5% |
| Lv2 | 10% |
| Lv3（μ） | 35% |
| Lv4 | 25% |
| Lv5 | 15% |
| Lv6 | 7% |
| Lv7 | 3% |

### 2.4 等級與檔位對照（墟界地圖分佈）

墟界目前115隻怪物，等級分佈如下：

| 怪物等級區間 | 數量 | 建議捕捉μ值 | 代表性地點 |
|------------|------|------------|---------|
| Lv1-8 | 41隻 | μ=3~5 | 新手野外（迷霧城外圍） |
| Lv9-18 | 33隻 | μ=10~15 | 初級副本（虛空裂縫） |
| Lv19-28 | 25隻 | μ=20~25 | 中級副本（迷失森林深處） |
| Lv29-38 | 14隻 | μ=32~35 | 高級禁地（試煉之塔） |
| Lv39-51 | 7隻 | μ=42~48 | 頂級禁地（碎影深淵） |

### 2.5 稀有寵物「1級」捕捉機制

| 條件 | 數值 |
|------|------|
| 捕捉到 Lv1 的整體機率 | **0.8%**（所有捕捉中） |
| Lv1 寵物為「稀有個體」的機率 | **50%**（進入檔位判定前） |
| 稀有個體的檔位 | **固定 S 檔（200+ BP）或 A 檔（150-199 BP）** |
| 稀有個體額外加成 | 自帶**2格天生技能**（非稀有僅1格） |

```
整體稀有寵物（S/A檔Lv1）出現率：0.8% × 50% = 0.4%
相當於每250次捕捉，期望出現1隻
```

## 三、五維屬性系統（BP檔位）

### 3.1 BP 五維說明

每隻寵物有五項 BP（Base Point），決定其成長潛力：

| BP | 影響屬性 | 每點BP帶來的效果 |
|----|---------|----------------|
| **體質** | HP上限、MP上限 | HP+8, MP+1, ATK+0.1, DEF+0.1, SPD+0.1 |
| **力量** | 物理攻擊力 | HP+2, MP+2, ATK+2.7, DEF+0.3, SPD+0.2 |
| **防禦** | 物理防禦力 | HP+2, MP+2, ATK+0.3, DEF+2.7, SPD+0.2 |
| **敏捷** | 速度、閃避、命中 | HP+3, MP+2, ATK+0.3, DEF+0.3, SPD+2.0 |
| **魔力** | 魔法攻擊、魔法治療量 | HP+1, MP+10, ATK+0.2, DEF+0.2, SPD+0.1 |

### 3.2 檔位（S/A/B/C/D/E）等級分級

| 檔位 | 俗稱 | BP總和範圍 | 整體佔比 | 抓到時加成 |
|------|------|-----------|---------|-----------|
| **S** | 傳說 | 200+ BP | 0.5% | +30% 屬性成長 |
| **A** | 稀有 | 150-199 BP | 9.5% | +20% 屬性成長 |
| **B** | 精良 | 110-149 BP | 25% | +10% 屬性成長 |
| **C** | 普通 | 70-109 BP | 45% | 標準成長 |
| **D** | 較差 | 30-69 BP | 17% | -10% 屬性成長 |
| **E** | 廢物 | 0-29 BP | 3% | -20% 屬性成長 |

> 💡 **Lv1稀有寵物特殊規則**：50%固定為S檔（200+ BP），50%固定為A檔（150-199 BP），不套用上述比例。

### 3.3 各檔位抓到時的五維BP隨機範圍

每項維度在該檔位範圍內均勻隨機分配（總和不得超過檔位上限）：

| 檔位 | 體質BP | 力量BP | 防禦BP | 敏捷BP | 魔力BP |
|------|--------|--------|--------|--------|--------|
| **S** | 40-80 | 40-80 | 40-80 | 40-80 | 40-80 |
| **A** | 30-60 | 30-60 | 30-60 | 30-60 | 30-60 |
| **B** | 18-45 | 18-45 | 18-45 | 18-45 | 18-45 |
| **C** | 10-30 | 10-30 | 10-30 | 10-30 | 10-30 |
| **D** | 3-18 | 3-18 | 3-18 | 3-18 | 3-18 |
| **E** | 1-10 | 1-10 | 1-10 | 1-10 | 1-10 |

### 3.4 升級BP成長計算

每升1級，系統給予：**3點固定BP + 1點隨機BP**（性格/成長型態加成額外計算）：

| 成長型態 | 說明 | 額外加成 |
|----------|------|---------|
| 鬥爭型 | 偏好攻擊 | 力量 +1/級 |
| 守護型 | 偏好防禦 | 防禦 +1/級 |
| 疾風型 | 偏好速度 | 敏捷 +1/級 |
| 魔力型 | 偏好魔法 | 魔力 +1/級 |
| 均衡型 | 全面發展 | 隨機維度 +1/級 |

```
離線掛機特殊規則：
- 離線期間僅獲得固定3點BP，不獲得隨機BP
- 確保長期玩家有優勢，但不明顯拉開差距（半放置平衡設計）
```

### 3.5 檔位對最終戰力的影響（實例計算）

以Lv50「均衡型」寵物為例，比較S檔 vs C檔抓到時的差距：

**抓到時（Lv5，假設均衡分配）**
- S檔抓到：總BP≈200，體/攻/防/敏/魔≈各40
- C檔抓到：總BP≈80，體/攻/防/敏/魔≈各16

**升級至Lv100（99級，共獲得495 BP）**
- S檔最終：總BP≈695
- C檔最終：總BP≈575（-17%）

**實例HP對比（S檔 vs C檔，均衡型）：**

| 等級 | S檔 HP | C檔 HP | 差距 |
|------|--------|---------|------|
| Lv1（抓到） | ~400 | ~160 | +150% |
| Lv30 | ~2,200 | ~1,100 | +100% |
| Lv60 | ~4,800 | ~2,700 | +78% |
| Lv100 | ~7,200 | ~4,400 | +64% |

## 四、寵物技能系統

### 4.1 技能整體架構

```
┌─────────────────────────────────────────────────────┐
│              寵物技能體系                              │
├─────────────────────────────────────────────────────┤
│                                                       │
│  【天生技能】  3格（捕捉時固定，Lv20/Lv50各解鎖1格）         │
│  ├─ 技能為寵物「種族固定」，不可刪除、不可覆蓋              │
│  └─ 定義該寵物的核心戰鬥特性                               │
│                                                       │
│  【天命技能】  3格（Lv15/Lv35/Lv60各解鎖1格）             │
│  ├─ 從墟界人物技能庫（32種）中篩選14種寵物適用子集         │
│  ├─ 技能可刪除、可替換                                    │
│  ├─ 技能可升級（Lv1-10，任務制）                          │
│  └─ 這是寵物差異化的核心                                   │
│                                                       │
│  合計：3天生 + 3天命 = 6格技能                            │
└─────────────────────────────────────────────────────┘
```

### 4.2 天生技能（3格固定欄位）

| 欄位 | 解鎖等級 | 說明 |
|------|---------|------|
| 天生技能格1 | 捕捉時自帶 | 種族核心技能，定義寵物基本定位 |
| 天生技能格2 | Lv20 自動習得 | 種族中階技能 |
| 天生技能格3 | Lv50 自動習得 | 種族終極技能 |

**天生技能特色**：
- 不可刪除、不可覆蓋
- 捕捉時隨機獲得該種族天技能庫中的1-3個（普通個體1個，稀有個體2個）
- 天生技能池大小：每種怪物設計3-5個天生技能選項

### 4.3 天命技能（14種寵物適用子集）

從墟界32種人物技能中，經戰鬥適用性篩選後，寵物可學的天命技能共**14種**：

**物理戰鬥系（4種）**

| 技能名 | 效果摘要 | 適合類型 |
|--------|---------|---------|
| 連擊 | 短時間連續攻擊2次，總傷110% | 物理寵 |
| 反擊 | 受攻擊時30%機率反擊一次 | 坦克寵 |
| 崩擊 | 對防禦姿態敵人造成+50%傷害 | 物理寵 |
| 毒擊 | 攻擊附加中毒（每回合HP-5%，持續3回合） | 干擾寵 |

**魔法攻擊系（4種）**

| 技能名 | 效果摘要 | 適合類型 |
|--------|---------|---------|
| 火焰魔法 | 火屬性法術，ATK×1.5 + MAG×2 | 魔攻寵 |
| 冰凍魔法 | 冰屬性法術，ATK×1.5 + MAG×2，20%減速 | 魔攻寵 |
| 風刃魔法 | 風屬性法術，ATK×1.5 + MAG×2，無視20%防禦 | 魔攻寵 |
| 隕石魔法 | 全體攻擊，ATK×1.0 + MAG×2.5 | 魔攻寵 |

**狀態控制系（5種）**

| 技能名 | 效果摘要 | 適合類型 |
|--------|---------|---------|
| 昏睡魔法 | 目標入睡2回合，被攻擊則驚醒 | 控制寵 |
| 石化魔法 | 目標石化1回合（無法行動） | 控制寵 |
| 混亂魔法 | 目標攻擊隨機目標1回合 | 控制寵 |
| 中毒魔法 | 目標中毒，每回合HP-8%，持續4回合 | 干擾寵 |
| 遺忘魔法 | 目標50%機率技能失效1回合 | 控制寵 |

**戰鬥輔助系（5種）**

| 技能名 | 效果摘要 | 適合類型 |
|--------|---------|---------|
| 補血魔法 | 恢復目標HP（MAG×3） | 輔助寵 |
| 氣絕回復 | 復活已倒下角色，恢復30%HP | 輔助寵 |
| 攻擊吸收 | 受到物理攻擊時，吸收30%轉為HP | 坦克寵 |
| 魔法吸收 | 受到魔法攻擊時，吸收30%轉為HP | 坦克寵 |
| 明鏡止水 | 清除自身所有負面狀態，恢復20%HP | 全能寵 |

> 排除：乾坤一擲（犧牲命中率，寵物AI不適用）、迅速果斷（需手動判斷先手）、亂射（範圍判定複雜）、酒醉（需飲食條件）、生產系（寵物不可學）

### 4.4 天命技能學習方式

| 方式 | 說明 | 限制 |
|------|------|------|
| **戰鬥自動領悟** | 攜帶天命技能上陣，每次出戰5%機率「頓悟」，自動學會該寵物可學的天命技能之一 | 每人每天最多領悟1次；僅從已解鎖列表中隨機 |
| **NPC學習** | 主城「獸語者」NPC，花費遊戲幣學習指定天命技能 | 每次學習100,000金；技能可重學（覆蓋原技能）|
| **技能書道具** | 野外BOSS/活動掉落「天命技能書」，使用後直接學會指定技能 | 為消耗品；同一技能書不可重複使用 |

### 4.5 天命技能升級（Lv1-10 任務制）

每場戰鬥結算時，統計各天命技能的**使用次數**，累積達標後自動升級：

| 等級 | 累積使用次數 | 效果增幅 |
|------|-----------|---------|
| Lv1 → Lv2 | 50次 | +15% |
| Lv2 → Lv3 | 120次（累計） | +15% |
| Lv3 → Lv4 | 250次（累計） | +15% |
| Lv4 → Lv5 | 450次（累計） | +15% |
| Lv5 → Lv6 | 700次（累計） | +15% |
| Lv6 → Lv7 | 1,000次（累計） | +15% |
| Lv7 → Lv8 | 1,500次（累計） | +15% |
| Lv8 → Lv9 | 2,200次（累計） | +15% |
| Lv9 → Lv10 | 3,000次（累計） | +20%（最終覺醒） |

> **半放置節奏參考**：以每天自動戰鬥100場估算，Lv5約需7天，Lv10約需30天。

**Lv10最終覺醒附加效果**：
- 連擊：2次 → 3次
- 補血魔法：附加清除1個負面狀態
- 攻擊吸收/魔法吸收：吸收率 30% → 45%
- 明鏡止水：附加恢復20%MP
