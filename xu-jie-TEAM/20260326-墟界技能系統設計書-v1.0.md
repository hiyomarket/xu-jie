# 墟界 ── 技能系統設計文件

> 設計日期：2026.03.26｜版本：v1.0｜狀態：初稿｜作者：大管家統合（遊戲企劃 + 企劃顧問共同建議）

---

## 一、設計背景與目標

### 參考標的
魔力寶貝（CrossGate）經典技能系統，並針對墟界進行簡化與現代化。

### 墟界的核心調整原則
| 魔力寶貝設計 | 墟界調整 |
|------------|---------|
| 職業綁定技能學習 | **無職業限制，任何人可學任何技能** |
| 等級門檻 | **廢除五元素/等級作為前提** |
| 任務門檻 | **保留，作為核心學習條件** |
| 技能格子上限 | 保留，**初期 8 格，可擴充至 16 格** |
| 技能 JP 系統 | 轉化為「**熟練度**」系統 |
| 職業試煉 | 轉化為「**挑戰任務**」 |
| 生產技能 30+ 種 | 精簡至 **3 大類** |

## 二、技能分類架構

### 五大類技能

| 類別 | 說明 | Lv 範圍 | 典型技能 |
|------|------|---------|---------|
| ⚔️ 攻擊技能 | 屬性魔法 + 物理輸出 | Lv1~Lv10 | 火球術、地裂術、連擊、乾坤一擲 |
| 🛡️ 防禦/輔助 | 恢復、反彈、無效、護盾 | Lv1~Lv10 | 生命恢復、魔法無效、反彈壁 |
| 🌙 狀態技能 | 控場（昏睡/石化/混亂/魅惑） | Lv1~Lv10 | 昏睡術、石化和亂咒 |
| ✨ 被動技能 | 常駐效果（吸收/加成） | Lv1~Lv10 | 攻擊吸收、魔法吸收、暴擊精通 |
| 🔨 生產技能 | 採集/製作（輕量保留） | Lv1~Lv5 | 基礎採集、簡易鍛造 |

## 七、生產技能（3 大類，簡化版）

| 類別 | 包含內容 | 取得方式 |
|------|---------|---------|
| **採集系統** | 伐木、採礦、採草（3 選 1 或全學） | 向 NPC 領取工具後，累計採集 50 次自動解鎖 |
| **製作系統** | 武器/防具/消耗品（3 個分支） | 持有對應素材 + 完成「新手工匠」任務 |
| **鑑定系統** | 單一「鑑定」技能 | 累計使用未鑑定道具 20 件 |

### 生產技能不納入的項目（建議刪除）
- ❌ 造槍、造斧、造杖、造弓（小刀/回力標）→ 統一為「武器製作」
- ❌ 製頭盔、製鎧甲、製袍、靴、鞋 → 統一為「防具製作」
- ❌ 武器修理、防具修理 → 統一為「維修」技能
- ❌ 變身、刻印 → 廢除或改為外觀系統

## 八、任務門檻設計

### 8.1 任務類型與難度對照

| 任務類型 | 難度 | 適用 Lv | 範例 |
|---------|------|---------|------|
| 📦 收集任務 | ⭐ | Lv1~Lv3 | 收集 10 個某魔物掉落物 |
| ⚔️ 戰鬥任務 | ⭐~⭐⭐⭐ | Lv2~Lv6 | 擊敗 30 隻某魔物 |
| 🧩 挑戰任務 | ⭐⭐~⭐⭐⭐⭐ | Lv5~Lv10 | 通關迷宮、1v1 對決 |
| 📜 故事任務 | ⭐⭐~⭐⭐⭐ | Lv3~Lv7 | 完成特定主線章節 |
| 🤝 聲望任務 | ⭐⭐~⭐⭐⭐ | Lv4~Lv8 | 某陣營聲望達到「尊敬」 |

### 8.2 任務難度與技能 Lv 對照

```
技能 Lv1  → 完成 1 個 ⭐ 收集/戰鬥任務
技能 Lv2  → 完成 1 個 ⭐⭐ 收集/戰鬥任務
技能 Lv3  → 完成 1 個 ⭐⭐ 戰鬥/故事任務
技能 Lv4  → 完成 1 個 ⭐⭐⭐ 挑戰任務
技能 Lv5  → 完成 ⭐⭐⭐ 挑戰 + 聲望門檻
技能 Lv6  → 完成 1 個 ⭐⭐⭐⭐ 挑戰任務
技能 Lv7  → 完成 ⭐⭐⭐⭐ 挑戰 + 故事任務
技能 Lv8  → 完成 ⭐⭐⭐⭐⭐ 限時挑戰
技能 Lv9  → 完成 ⭐⭐⭐⭐⭐ 挑戰 + 多條件
技能 Lv10 → 完成 ⭐⭐⭐⭐⭐ 終極挑戰（每人僅能解 1 個）
```

## 九、技能學習流程

### 9.1 標準學習流程

```
玩家抵達導師 NPC
       │
       ▼
  查詢技能列表 → 看見【技能 A】條件：「完成 ⭐⭐ 任務《○○》
       │
       ▼
  接受任務 → 完成戰鬥/收集/挑戰
       │
       ▼
  導師確認 → 支付金幣/魂晶 → 習得 Lv1
       │
       ▼
  使用技能累積熟練度 → 自動升級 Lv2~Lv5
       │
       ▼（Lv5 以上）
  接受挑戰任務 + 材料 → Lv6~Lv9
       │
       ▼（Lv10）
  通過試煉 → 習得終極技能
```

### 9.2 技能學習費用（建議）

| 技能 Lv | 貨幣 | 金額/數量 |
|--------|------|---------|
| Lv1 | 金幣 | 500 |
| Lv2 | 金幣 | 1,200 |
| Lv3 | 金幣 | 2,500 |
| Lv4 | 金幣 | 5,000 |
| Lv5 | 金幣 + 魂晶 | 10,000 + 5 |
| Lv6 | 魂晶 | 20 |
| Lv7 | 魂晶 | 50 |
| Lv8 | 魂晶 | 120 |
| Lv9 | 魂晶 | 300 |
| Lv10 | 魂晶 + 稀有道具 | 500 + 稀有道具 1 個 |

### 9.3 NPC 角色定義

| NPC 類型 | 職責 |
|---------|------|
| **技能導師** | 給予任務、教授技能（每位導師負責 2~3 類技能） |
| **試煉管理者** | 專門處理 ⭐⭐⭐ 以上挑戰型任務的驗證與發放 |
| **商人 NPC** | 販售技能升級素材（非任務型技能強化） |

### 9.4 資訊揭露機制

| 途徑 | 機制 |
|------|------|
| **技能列表 UI** | 滑鼠懸停顯示「習得條件：完成 ★ 任務《○○》」 |
| **導師 NPC** | 對話時自動列出「可學習技能」與「未解鎖技能」的任務條件 |
| **圖鑑/百科** | 記載所有技能與其習得任務，方便玩家規劃 |
| **引導任務** | 新手期給予第一個技能任務，作為學習範本 |

## 十、技能升級機制

### 10.1 Lv1~Lv5：使用次數累積制

每施放/使用技能 1 次，累計熟練度 +1，自動升級，無需回 NPC。

| 技能 Lv | 累積使用次數 | 所需時間參考（每場戰鬥用 3 次為例） |
|---------|------------|-------------------------------|
| Lv1 → Lv2 | 50 次 | ~17 場戰鬥 |
| Lv2 → Lv3 | 100 次（累計 150）| ~50 場戰鬥 |
| Lv3 → Lv4 | 150 次（累計 300）| ~100 場戰鬥 |
| Lv4 → Lv5 | 200 次（累計 500）| ~167 場戰鬥 |

### 10.2 Lv5~Lv10：大師之路（階段性任務制）

| 等級 | 條件 |
|------|------|
| Lv5 | 累計使用 500 次（自動） |
| Lv6 | 完成 1 次對應的「突破任務」 |
| Lv7 | 累計造成/治療特定數值 + ⭐⭐⭐⭐ 挑戰 |
| Lv8 | 完成「資深試煉」地下城 |
| Lv9 | PvP 場合使用該技能並取勝 + 多條件 |
| Lv10 | 通過終極試煉（每人僅能解 1 個 Lv10 技能） |

### 10.3 輔助升級管道（避免純農）

| 管道 | 效果 |
|------|------|
| 🧪 **技能結晶（道具）** | 使用後直接獲得 30 次熟練度；可從副本/BOSS 掉落 |
| 📚 **導師特訓（每週限 1 次）** | 支付金幣，導師陪你演練 50 次，快速累積熟練度 |
| ⚔️ **挑戰通關加成** | 完成對應難度挑戰後，該技能熟練度 +100（一次性） |

### 10.4 技能忘却機制

- 忘記技能可退還 **50% 熟練度**
- 重新學習需重新完成任務門檻
- 防止玩家亂點，鼓勵策略性配置

## 十一、試煉系統（Lv10 專屬）

### 終極技能（Lv10）必須通過「試煉」才能解鎖。

```
試煉類型：
├── 🗡️ 單人試煉：限時內擊敗特定 BOSS
├── 🧙 知識試煉：回答與世界觀相關的問題（5 題對 4 題過）
├── 🤝 協助試煉：幫助其他玩家完成任務 N 次
└── 🌟 綜合試煉：結合上述三項的複合挑戰
```

## 十二、與魔力寶貝的差異總覽

| 項目 | 魔力寶貝 | 墟界 |
|------|---------|------|
| 技能學習限制 | 職業綁定 | **無職業限制，純任務門檻** |
| 學習前提 | 等級 + 職業 + 試煉 | **僅任務條件** |
| 五元素前提 | 有 | **無** |
| 技能格子上限 | 10 格 | **初期 8 格，可擴充至 16 格** |
| 生產技能數量 | 30+ 種 | **精簡至 3 大類** |
| 技能忘卻 | 有 | **有（退還 50% 熟練度）** |
| 試煉系統 | 職業試煉 | **挑戰任務取代** |
| JP 系統 | Job Point | **熟練度（使用次數）** |

## 十三、系統特色摘要

| 特色 | 說明 |
|------|------|
| **無職業牆** | 任何技能只要完成任務即可學 |
| **任務即門檻** | 取代職業綁定，讓玩家有目標感 |
| **使用即成長** | Lv1~Lv5 靠使用次數自動升級，減少 NPC 依賴 |
| **挑戰區分大師** | Lv6~Lv10 透過任務質量區分普通玩家與高手 |
| **試煉終極化** | Lv10 技能必須通過試煉，極具成就感 |
| **生產系輕量化** | 從 30+ 種精簡至 3 大類，降低系統負擔 |
| **社交鼓勵** | 治療系以「協助他人」為升級條件 |
