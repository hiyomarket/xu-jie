## 會員系統優先度

| 功能 | 優先度 | 說明 |
|------|--------|------|
| 會員註冊/登入（含 OAuth）| **P0** | 玩家進不來後面都免談 |
| 用戶資料 CRUD | **P0** | 後台基本功能 |
| 數據儀表板（DAU/營收）| **P1** | 營運決策依據 |
| 道具/怪物參數調整 | **P1** | 上線後平衡必要 |
| 客服工具 | **P2** | 上線後再補 |

## ⚠️ 風險提示

### 1. 資安風險
- Discord OAuth 需妥善處理 token
- Refresh Token 需安全儲存

### 2. 擴展性風險
- 未來玩家增長需考慮資料庫分庫（目前建議：單一 MySQL，先求有）
