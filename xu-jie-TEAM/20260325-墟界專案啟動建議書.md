## 二、已確認的核心遊戲機制

### 2.1 四大元素系統（火/風/地/水）
```
火 → 風 → 地 → 水 → 火（循環剋制）
剋制倍率：+30%（直接採用魔力寶貝原版數值）
```
**資料來源**：魔力寶貝網路公開公式，可直接引用，不需要重新平衡。

### 2.2 角色屬性計算
- 直接參考魔力寶貝原版公式（網路可查）
- 不自行發明，確保數值平衡有據可循

### 2.3 抓寵系統（簡化版封印捕捉）
| 條件 | 設計內容 |
|------|---------|
| 捕捉前提 | 戰鬥中遭遇過該魔物（即「遇見過」）|
| 血量門檻 | 血量 ≤ 30% 時捕捉率最高 |
| 捕捉器等級 | 1-5級（免費/遊戲幣/月卡/VIP差異）|
| 變異寵 | VIP專屬捕捉器，變異寵機率 +20% |
| 寵物檔數 | 每隻寵物有檔位（資質），影響成長上限 |

### 2.4 經濟三層制
| 貨幣 | 定位 |
|------|------|
| 靈石 | Premium，台幣付費 |
| 遊戲幣 | 遊戲內產出 |
| 積分值 | 免費/付費混合 |

### 2.5 核心遊戲循環
```
每日登入 → 領取離線收益 → 消耗體力推圖
→ 打怪掉寶 → 練寵抓寵 → 拍賣行交易 → 提升戰力
```

## 三、會員系統設計（遊戲企劃提供）

### 三層會員制

| 層級 | 創角上限 | 寵物欄 | 捕捉器 | 離線收益加成 | 特色功能 |
|------|---------|--------|--------|------------|---------|
| **免費玩家** | ×1 | 5格 | 基礎捕捉器 | 標準 | 四大元素任務線 |
| **月卡會員** | ×3 | 15格 | 進階捕捉器 | +50% | 任務加成經驗 |
| **尊榮會員** | ×5 | 30格 | VIP捕捉器 | +100% | 變異寵+20%、專屬稱號 |

**設計意圖**：降低免費門檻體驗核心循環，月卡作為輕度付費切入點，尊榮滿足深度玩家歸屬感。

## 四、商業模式建議（市場分析師提供）

### 4.1 收入來源結構（參考）
| 來源 | 說明 | 佔比目標 |
|------|------|---------|
| 月卡會員 | 連續訂閱，每月收費 | 50% |
| 一次性內購 | 寵物、道具、禮包 | 30% |
| 拍賣行手續費 | 交易徵稅 | 20% |

### 4.2 便利型付費項目（建議）
| 付費項目 | 功能 |
|---------|------|
| 高級捕捉器 | 捕捉成功率 +15% |
| 離線收益加速券 | 2x / 4x 加速 |
| 一鍵練功 | 自動戰鬥加速 |
| 寵物檔數鎖定 | 洗到好檔後鎖定，不再重來 |

### 4.3 ❌ 不建議的變現模式
- **純訂閱制**：排除免費玩家，口碑傳播受限
- **廣告變現**：體驗破壞，放置玩家對廣告敏感度極高

### 4.4 ⚠️ Boss 最新裁決
> **不做收費制，游戲建立起來最重要，付費機制日後再定。**

## 五、技術架構（架站顧問提供）

### 5.1 技術棧建議
| 層次 | 建議技術 | 說明 |
|------|---------|------|
| **前端框架** | React 19 | 與天命共振一致 |
| **後端框架** | NestJS + tRPC | 類型安全，長期維護性佳 |
| **資料庫** | MySQL（在 Vultr VPS 上）| 遊戲數據適合關聯式資料庫 |
| **認證** | JWT（遊戲）+ Session（後台）| 遊戲 client 用 JWT，RMS 後台用 Session |
| **部署** | Docker 容器化 | 與 hiyo-photo-backend 隔離 |

### 5.2 部署架構
```
Cloudflare CDN ──▶ Nginx (Port 80/443)
                      │
          ┌───────────┴───────────┐
          ▼                       ▼
   oracleres-server       xujie-server
   (Port 3001)            (Port 3000)
                                  │
                              MySQL
                           (Port 3306)
```

### 5.3 Vultr VPS 建議
| 項目 | 規格 |
|------|------|
| **建議規格** | 4 vCPU / 8GB RAM / 80GB SSD |
| **預估費用** | 約 $48/月 |
| **機房** | 新加坡（亞洲延遲最低）|
| **作業系統** | Ubuntu 22.04 LTS |
| **額外** | Docker 容器化，自動備份每週 |

### 5.4 SSL 憑證
