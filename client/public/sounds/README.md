# 墟界遊戲音樂素材

## 說明

此目錄用於存放遊戲背景音樂。

## 放置音樂檔案

請將以下格式的音樂檔案放置到此目錄：
- `town.mp3` - 主城/城鎮背景音樂
- `battle.mp3` - 戰鬥背景音樂
- `menu.mp3` - 選單背景音樂

## 音樂格式

- 格式：MP3, OGG, 或 WAV
- 建議：128kbps, 44100Hz

## 放置後

放置音樂檔案後，在 `src/app/layout.tsx` 中取消註釋音樂元件即可啟用。

## 版權建議

- 使用免費版權音樂（CC0 / Royalty Free）
- 或使用 AI 生成音樂
- 或請美術顧問協助製作
